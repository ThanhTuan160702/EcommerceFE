# A fullstack ecommerce with react, express, mongodb, nodejs

Live demo:
https://ecommerce-z.vercel.app/

## Features
- Sign-In, Sign-Up, Forgot password
- Manage product
- Filter, sort product
- Manage user
- Manage order
- Paypal
